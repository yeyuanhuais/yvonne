闭包（closure）是一个函数以及其捆绑的周边环境状态（lexical environment，词法环境）的引用的组合。换而言之，闭包让开发者可以<Highlight>从内部函数访问外部函数的作用域</Highlight>。在 JavaScript 中，闭包会随着函数的创建而被同时创建。


> 来源：https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Closures
