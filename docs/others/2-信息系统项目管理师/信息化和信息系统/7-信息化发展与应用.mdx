## 7.3. 电子政务
:::note
电子政务与传统政务相比，在办公手段、业务流程以及与公众沟通的方式上都存在很大区别

电子政务并不是要完全取代传统政务、也不是简单的将传统政务原封不动的搬到Internet上
:::

### 7.3.1. 电子政务的应用模式
1. 政府对政府（G2G）
2. 政府对企业（G2B）
3. 政府对公众（G2C）
4. 政府对公务员（G2E）

### 7.3.2. 国家电子政务总体框架
国家电子政务总体框架的构成包括：服务与应用系统、信息资源、基础设施、法律法规与标准化体系、管理体制；

推进国家电子政务建设，服务是宗旨，应用是关键，信息资源开发利用是主线，基础设施是支撑，法律法规、标准化体系、管理体制是保障。

## 7.4. 电子商务
基本特征：
1. 普遍性：将生产企业、流通企业、消费者以及金融企业和监管者集成到了数字化的网络经济中
2. 便利性：参与电子商务的各方不受地域、环境、交易事件的限制
3. 整体性：将人工操作和电子信息处理集成为一个不可分割的整体
4. 安全性：必须采取诸如加密、身份认证、防入侵、数字签名、防病毒等技术手段确保交易活动的安全性
5. 协调性：各方需要有序的协作，共同配合来完成交易

### 7.4.1. 电子商务类型
1. 按照依托网络类型划分：EDI（电子数据交换）商务、Internet（互联网）商务、Intranet（企业内部网）商务和Extranet（企业外部网）商务
2. 按照交易对象划分：B2B模式（企业与企业之间）、B2C模式（企业和消费者个人之间）、C2C模式（消费者和消费者之间）、O2O模式（线上购买线下的商品和服务，实体店提货或者享受服务）；B2G（企业对政府）、C2B（消费者对企业）、C2G（消费者对政府）

### 7.4.2. 电子商务的基础设施
网络基础设施、多媒体内容和网络出版的基础设施、报文和信息传播的基础设施、商业服务的基础设施

### 7.4.3. 电子商务建立标准的领域
1. 用户接口
2. 基本功能
3. 数据及客体

## ★ 7.5. 工业和信息化融合
:::note
实施“中国制造2025”，促进两化融合，加快从制造大国转向制造强国，需要电子信息产业有力支撑，大力发展新一代信息技术，加快发展智能制造和工业互联网；制定“互联网+”行动计划，推动移动互联网、云计算、大数据、物联网等应用，需要产业密切跟踪信息技术变革趋势，探索新技术、新模式、新业态，构建以互联网为基础的产业新生态体系
:::

:::tip
工业化与信息化“两化融合”的含义：

4. 信息技术与工业技术、IT设备与工业装备的融合
:::

### 7.5.1 企业信息化以及企业信息系统的目标
- 信息化是指充分利用信息技术改造、推动、提升和发展各个行业，造福于人类社会的过程。
- 企业信息化，就是企业将信息技术手段应用到企业的生产和运营管理中，利用信息技术来改造和提升自己业务和管理水平的过程。
- 信息化的过程是通过一个个的项目来完成的。这些项目建成的信息系统作为各行业、各单位的数字神经系统，支持和推动相关行业和相关单位的发展。
- 信息系统的功能就在服务各行业、各单位业务发展的过程中得以体现，如一个企业的MRP系统，它的功能不仅要支持企业的进销存业务，还要支持生产和财务，根据具体企业的需要MRP还可以具有其他的一些功能，同时MRP要求各部门各业务的数据统一和集中。
- 因此，企业信息系统不仅支持企业的单个业务，集成的企业信息系统也支持企业的业务协同，进而支持企业的信息共享。

:::tip
- 数据标准化是信息化的一部分工作，是企业信息系统的基础
- 企业需求分析是建设企业信息系统的一个早期阶段
- 企业信息系统又助于技术提升
- 信息共享和业务协同是企业信息系统的重要目标
:::

### 7.5.2. 企业信息化发展的战略要点
1. 以信息化带动工业化
2. 信息化与企业业务全过程的融合、渗透
3. 信息产业发展与企业信息化良性互动
4. 充分发挥政府的引导作用
5. 高度重视信息安全
6. 企业信息化与企业的改组改造和形成企业制度有机结合
7. “因地制宜”推进企业信息化

## 7.6. 智慧化
## ★ 7.6.1. 智能的特点
1. 具有感知能力，即具有能够感知外部世界、获取外部的信息的能力，这是产生智能活动的前提条件和必要条件
2. 具有记忆和思维能力，即能够存储感知到的外部信息及思维产生的知识，同时能够利用已有的知识对信息进行分析、计算、比较、判断、联想、决策
3. 具有学习能力和自适应能力，即通过与环境的相互作用，不断学习积累知识，使自己能够适应环境变化
4. 具有行为决策能力，即对外界的刺激作出反应，形成决策并传达相应的信息

## 十四五

:::tip
《“十四五”推进国家政务信息化规划》明确主要目标：提出到2025年，政务信息化建设总体迈入以数据赋能、协同治理、智慧决策、优质服务为主要特征的融慧治理新阶段
:::

我国实现社会主义现代化，“新四化”的同步发展是基本路径，也是重要目标。我们将持续推动新型工业化、信息化、城镇化、农业现代化同步发展，到2035年基本实现“新四化”

- 5G用户普及率提高到56%，建设E级和10E级超级计算中心；
- 数字技术聚焦高端芯片、操作系统、人工智能关键算法、传感器等关键领域；
- 加快布局量子计算、量子通信、神经芯片、DNA存储等前沿技术；
- 实施"上云用数赋智"行动，推动数据赋能全产业链协同转型；
- 到2025年，前沿数字技术创新生态体系日益完备，行业级融合应用示范标杆不断涌现；
- 超融合基础架构HC1,多套单元设备的网络聚合，无缝横向扩展，形成统一的资源池；
- 边缘计算，提供最近端服务；数字孪生，推动城市空间摆脱物理约束，进入数字空间；
- 超级应用，不仅有自己的一套功能，还为第三方提供微应用的平台；元宇宙的核心技术：
- 扩展现实技术、数字孪生和区块链；泛在智能，跨越不同环境，以简单、自然的对话方式处理信息，享受服务；
- 6G,数据传输速率可能达到5G的50倍，时延缩短5G的十分之一；
- 类脑计算，又称神经形态计算，借鉴生物神经系统相对处理模式；
- 脑机接口，又称大脑端口或脑机融合感知，连通人或动物脑与外部设备；
- 零信任网络安全，创建数据为中心的边界，强身份验证；东数西算，数为数据，算为算力，将东部算力有序引导到西部；
- web3.0 打造一个去中心化的通用数字身份体系；数字藏品，应用了区块链技术

《“十四五”软件和信息技术服务业发展规划》，推进重点领域数字化发展。支持新一代信息技术在普惠金融领域创新应用，发展相关软件产品。培育物流运输、分拣、仓储、配送等各环节软件解决方案，提升物流数字化水平。推进交通软件应用，提高交通运输资源利用效率和管理精细化水平。支持城市信息模型、地理信息系统、建筑信息模型和建筑防火模拟等软件创新应用，实施智能建造能力提升工程，推进建筑业数字化、网络化、智能化突破。支持农业基础资源数据库、智能监测控制系统、农产品质量安全追溯系统等开发应用，强化综合信息服务，提高农业农村数字化水平。持续征集并推广智慧城市典型解决方案，支持城市大脑、精准惠民、智慧政务、城市体检等城市级创新应用，培育软件与智慧社会融合发展的新模式、新应用、新业态。数字城市是数字地球的重要组成部分，是传统城市的数字化形态。数字城市是应用计算机、互联网、智慧城市不但广泛采用物联网、云计算、人工智能、数据挖掘、知识管理、社交网络等技术工具，也注重用户参与、以人为本的创新2.0理念及其方法的应用，构建有利于创新涌现的制度环境，以实现智慧技术高度集成、智慧产业高端发展、智慧服务高效便民、以人为本持续创新，完成从数字城市向智慧城市的跃升。
