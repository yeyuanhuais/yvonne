### 调用示例
```javascript 
axios
  .post("url", {}, {
    onUploadProgress: (p) => {
      p.loaded != p.total && setLoadNum(parseInt(100 * (p.loaded / p.total)));
    },
    onDownloadProgress: (p) => {},
  })
```
